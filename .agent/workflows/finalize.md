---
description: Finalize and merge current feature branch
---

1. Check git status to see unstaged/staged changes.
2. Remove unnecessary files and temporary files.
3. Add all changes using `git add .`
4. Commit the changes using `git commit -m "[commit message]"`
5. Checkout the main branch using `git checkout main`
6. Merge the feature branch into main using `git merge [feature-branch-name]`
